user module

Este módulo proporciona un servidor Flask para la gestión de usuarios y de todos sus datos.

user.add_Resultado()

Agregar un nuevo análisis del seguimiento a la base de datos.

Parámetros:

  • Token (JWT) – Token de autentificación.

  • json (dict) – JSON con los datos del formulario que contiene: - x (int): Coordenada X de la lesión. - y (int): Coordenada Y de la lesión. - lugar (str): Lugar de la lesión en el cuerpo. - body (str): Lugar de la lesión en el cuerpo(frontal/trasera). - imagen (str): Imagen en Base64. - predicciones (str): Array con las predicciones.

Devuelve:

Un diccionario con el mensaje de éxito o error.

  • mensaje (str): Mensaje de éxito o error.

Tipo del valor devuelto:

dict

user.get_historial()

Conseguir el historial de una lesión de la base de datos.

Parámetros:

  • Token (JWT) – Token de autentificación.

  • json (dict) – JSON con los datos del formulario que contiene: - id (int): Identificador de la lesión.

Devuelve:

Una lista de diccionarios que contiene los detalles del historial.

Cada diccionario tiene las siguientes claves: - id (int): Identificador único. - subId (int): Identificador secundario. - dia (int): Día del evento. - mes (int): Mes del evento. - año (int): Año del evento. - imagen (str): Imagen en formato base64. - probabilidad (str): Probabilidad calculada. - predicciones (str): Lista de predicciones. - color (str): Color asociado a la predicción.

Tipo del valor devuelto:

list
